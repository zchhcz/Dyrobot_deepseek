## 完整时间线

### 前史
- **小学**：室内鞋被藏近60次（其中约50次是女生做的）。被取绰号「自恋谷」。高年级开始不再收到生日/圣诞礼物。
- **国中二年级**：担任班长 → 对女班长告白 → 「怎么可能，有够恶心的」→ 全班传开 → 彻底封死对异性的期望
- **高一入学第一天**：为救一只狗被豪华轿车撞 → 住院三周 → 左脚骨折 → 错过交友"黄金期" → 那只狗是由比滨结衣的 → 车里坐的是雪之下雪乃（命运的交织早在一年前就开始了）
- **高一某日**：写作文《高中生活回顾》成"犯罪宣言" → 被班主任平冢静逮住 → 强制送入侍奉社

### 高二篇（主线剧情）

**Vol.1 — 春天**
- 侍奉社初识雪之下雪乃 → 互相看不顺眼但能跟上对方思维
- 由比滨结衣第一次来侍奉社 → 烤焦的饼干 → 被雪之下毒舌后反而说"好帅气"
- 材木座义辉的轻小说咨询 → 八幡说「材木座不需要改变」

**Vol.2-3 — 春天到初夏**
- 川崎沙希的深夜打工问题 → 八幡式解决
- 由比滨开始叫八幡"自闭男"(ヒッキー)
- 八幡对雪之下的评价：「我能体会雪之下的孤独」

**Vol.5 — 夏天**
- 烟火大会 → 阳乃出场 → 八幡初次感受到雪之下家族的压力
- 暑假集训 → 由比滨知道八幡就是救她狗的人
- 八幡：「如果你是因为那样才对我好，请停下来」

**Vol.6 — 秋天（文化祭）**
- 文化祭执行委员会 → 相模南只想头衔不想做事
- 雪之下独自扛下所有工作直到病倒 → 八幡在空教室找到她
- 八幡用"最卑鄙的手段"煽动现场组罢工反弹 → 结构破坏式解决
- 文化祭最后的校庆活动 → 八幡与雪之下在空教室独处

**Vol.7 — 初冬（学生会长选举）**
- 学生会长选举 → 雪之下想参选 → 八幡不想让她参选 → 背后操纵选举
- 一色伊吕波当选 → 雪之下：「我很讨厌你的做法。虽然说不出为什么……总之我非常讨厌」
- 侍奉社会议陷入虚伪的和平 → 八幡、雪之下、由比滨三人都在逃避

**Vol.9 — 深冬（真物宣言）**
- 侍奉社气氛冰冷 → 每个人都在说无关痛痒的话
- 由比滨：「不说出口便不会了解」→ 八幡反驳：「说出口也不代表会了解」
- **真物宣言**（line 61356）：「我还是，想得到『真物』。」——八幡颤抖着，流着泪说出
- 雪之下：「我无法了解」→ 逃离 → 空中走廊 → 由比滨追上去抱住她
- 雪之下：「我接受你的委托。」——三人的关系进入新阶段

**Vol.10-11 — 冬季**
- 阳乃介入 → 称八幡和雪之下的关系为"共依存"
- 八幡陷入自我厌恶 → 开始质疑自己对雪之下的感情
- 迪士尼场景 → 雪之下：「总有一天，要来救我喔」——第一次对八幡说出愿望

**Vol.12 — 深冬（情人节）**
- 情人节前夕 → 雪之下第一次完整坦白自己的家庭和过去
- 八幡的动摇（line 69896）：「我并不想要冰冷残酷、只会带来悲伤的真物」
- 三人临海公园雪夜 → 沉默 → 一直存在的距离感

**Vol.13 — 冬末**
- 由比滨用泪水反驳阳乃的"共依存"论断
- 八幡决心行动 → 制造冒牌舞会给雪之下的真舞会当挡箭牌
- 对阳乃宣战：「会把名为真物的东西展现给你看」

**Vol.14 — 春天（告白的季节）**
- 联合毕业舞会 → 八幡与雪之下一起筹备
- 雪之下母亲认可 → 对八幡的态度从"需要处理的麻烦"变成"需要观察的对象"
- **阳台告白**（line 94611）：「我喜欢你。比企谷君。」→ 说完就跑 → 八幡：「那些麻烦得要死的地方却也可爱得要死」
- 小町入学总武高中 → 与一色联手复活侍奉社
- **最终话**（line 94840）：由比滨重新走入侍奉社 → 新委托：「我喜欢的人有了女朋友，她是我最好的朋友……该怎么办？」
- 雪之下拉开中间的椅子：「请坐吧。看起来要说很久呢」
- 由比滨：「……可能要很久。不只是今天就结束了，明天也，后天也……大概会一直持续下去」
- 最后一句：「——我的青春恋爱物语果然有问题。」

3. **方法受质疑期**（Vol.7-8）：「有人讨厌我的方法——而且我说不出反驳」
4. **渴望暴露期**（Vol.9）：「我想要的不是那种东西——我想要的是这个」
5. **恐惧与后退期**（Vol.10-12）：「真物带来的可能只有悲伤——我不想要」
6. **接受与持续期**（Vol.13-14）：「不是找到了——是接受了需要一直走下去」
